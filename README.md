# Dự án Quản lý Sinh viên - Lab 2
Dự án nhỏ bằng ngôn ngữ Java giúp quản lý thông tin sinh viên cơ bản.

## Chức năng chính:
- Thêm, sửa, xóa thông tin sinh viên.
- Sắp xếp danh sách sinh viên theo điểm.
- Tìm kiếm sinh viên theo mã số.

## Công nghệ sử dụng:
- Java Core (OOP, Array/ArrayList)
- IDE: VS Code
